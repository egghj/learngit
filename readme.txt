有人住高楼，有人在深沟，有人光万丈，有人一身锈，世人万千种，浮云莫去求，斯人若彩虹，遇上方知有。
老美的青梅竹马两小无猜拍的也能如此真实直击人心，
一个很简单的故事，每处的细节，男女主的内心活动，互相闹别扭互相和好，这种细微的变化处理的恰到好处。
在面对青少年早恋问题，中西方俨然是不同的态度，这样美好的故事，有一大部分的功劳是家庭教育，父母正确的引导，而不是一味的批评遏制，
青春期男女孩之间的战争也许就这么简单，一见钟情或怦然心动，而两个小演员也表演地非常自然。
再类比国内青春影视作品，负能量夸张成分更多，这也是为什么怦然心动能在国内大受欢迎的原因。

长久以来，网路盛行这样一句话：“斯人若彩虹，遇上方知有。”可是听来怎么也没有“斯人若彩虹，let it be me”这样洋气而又霸气。
想必这样的题目，这样的电影，能点开并且看到这里的人大多是心思绵密如丝缎的女孩们。
可是初恋这样的桥段，这样的故事，那样的悸动，那样的青春，思来无论是男孩还是女孩，看后都能感受到强烈的美好与温柔。
青梅竹马、两小无猜的故事演绎了一代又一代，走过了一国又一国。爱情是人类永恒的话题，而初恋则是人类共同的回忆。
看过太多充满遗憾的初恋故事，初恋两字，似乎就是“酸涩”的代名词。然而《Flipped》明亮清新的画面，甜蜜舒缓的节奏，让我们这些看过世间丑恶诡谲的成人似乎又找到了遗失的美好。
Julie是一个坚强、果敢、自立的女孩。
她可以挺身而出捍卫一棵梧桐树，
为了守住那一片“整体大于部分之和”的风景；她可以持之以恒六年去爱一个Bryce，
为了得到那一个殷殷切切梦寐许久的初吻。开始的六年间，她一直在主动，却也一直是被动的，
所以当Bryce在晚宴上患得患失地接近她时，她断然拒绝，却也掩饰不住地感到“我感觉自己掌握了主动，这感觉真好”。
Julie反反复复地提及Bryce的那双dazzling eyes，她沉陷于那样的眼眸之中不可自拔，她的怦然心动也是从这里开始。
然而在这之后的许多年间，她所以为的一见钟情却换回Bryce不停地躲避，她也渐渐开始思考这样的一双明眸到底意味着什么，这究竟是不是爱情？
整个过程在以一种真实的速度不紧不慢地展现了女主角的成长之路。
